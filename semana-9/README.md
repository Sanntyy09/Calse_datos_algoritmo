# 🌳 Semana 9 - Árboles y Simulación de Arenas

## 📌 Descripción

En esta semana se trabajó el concepto de **árboles en estructuras de datos**, comprendiendo su funcionamiento, recorrido y aplicación en problemas reales.

Además, se desarrolló una **simulación tipo “arenas”**, donde se aplican estructuras como colas y lógica de procesamiento para modelar comportamientos dinámicos.

---

## 🌳 Tema Principal: Árboles

Se estudiaron los siguientes conceptos:

* Definición de árbol binario
* Nodos (raíz, hijos, hojas)
* Recorridos:

  * Inorden
  * Preorden
  * Postorden
* Búsqueda en árboles:

  * DFS (Depth First Search)
  * BFS (Breadth First Search)

### 💡 Aplicación

Los árboles permiten organizar datos de forma jerárquica y son fundamentales en:

* Sistemas de búsqueda
* Bases de datos
* Inteligencia artificial

---

## 🏜️ Simulación de Arenas

Se implementó un sistema que simula el comportamiento de elementos (arenas) mediante estructuras de datos.

### 🔎 ¿De qué trata?

* Se manejan entradas de datos que representan eventos o acciones
* Se organizan mediante estructuras como colas
* Se procesan siguiendo un orden específico
* Se obtiene un resultado basado en reglas definidas

### 🎯 Objetivo

Modelar situaciones del mundo real donde múltiples elementos interactúan de manera ordenada y dinámica.
