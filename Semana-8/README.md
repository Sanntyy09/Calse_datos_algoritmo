# 🌳 Semana 8 - Árboles y Búsqueda Binaria

En esta semana se desarrollan conceptos fundamentales de estructuras de datos no lineales, enfocándose en la implementación de **Árboles Binarios de Búsqueda (BST)** y algoritmos de **búsqueda eficiente**.

---

## 🎯 Objetivo

Comprender el funcionamiento de los árboles binarios de búsqueda y aplicar operaciones como inserción, búsqueda y recorridos, optimizando el acceso a los datos.

---

## 📚 Temas Trabajados

- 🌳 Árboles binarios
- 🔍 Búsqueda binaria
- ➕ Inserción en BST
- 🔎 Búsqueda de elementos
- 🔁 Recorridos:
  - Preorden
  - Inorden
  - Postorden

---

## 🧠 Análisis de Complejidad

| Operación     | Mejor Caso | Caso Promedio | Peor Caso |
|--------------|-----------|--------------|-----------|
| Inserción     | O(log n)  | O(log n)     | O(n)      |
| Búsqueda      | O(log n)  | O(log n)     | O(n)      |
| Recorridos    | O(n)      | O(n)         | O(n)      |

📌 El peor caso ocurre cuando el árbol se desbalancea (similar a una lista).

---

## 📌 Conclusión

El uso de árboles binarios de búsqueda permite optimizar operaciones de búsqueda e inserción, aunque su eficiencia depende de la estructura del árbol.
